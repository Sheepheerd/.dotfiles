# Keymaps

## Non-Plugin Related
<details/>
    <summary>basic</summary>
- `<leader>t`   - Enter Terminal Mode
- `<Esc>`       - Escape Terminal Mode
- `<leader>d`   - Black Hole Delete
- `<leader>e`   - Vim File Explorer
</details>
